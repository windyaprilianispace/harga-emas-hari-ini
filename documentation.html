Dokumentasi API Harga Emas

API ini menyediakan data harga emas terbaru dalam berbagai satuan (contoh: ons, gram, kilogram) dari harga-emas.org.

Home Data Harga Emas Dokumentasi API

Overview

API Harga Emas memungkinkan pengguna untuk mengakses data harga emas secara langsung dalam USD dan IDR. Data diambil dari sumber eksternal dan dikembalikan dalam format JSON agar mudah diakses.

Base URL: https://your-api.vercel.app

Endpoints

1. Mendapatkan Harga Emas Saat Ini

Endpoint ini digunakan untuk mendapatkan harga emas terbaru.

Method: GET

URL: /api/gold-prices

Contoh Request:

GET https://your-api.vercel.app/api/gold-prices

Response

Response dikembalikan dalam format JSON. Berikut adalah contoh response yang akan diterima:

{
    "usd": {
        "oz": "1800",
        "gr": "57.87",
        "kg": "57870"
    },
    "kurs_bi": {
        "oz": "26000",
        "gr": "830.50",
        "kg": "830500"
    },
    "idr": {
        "oz": "400000",
        "gr": "12800",
        "kg": "12800000"
    },
    "update_gold_price": "2024-09-21",
    "update_kurs_bi": "2024-09-21",
    "source": "https://harga-emas.org",
    "credit": "Powered by Windy, creator of this API",
    "website": "https://your-website.com"
}

Response ini mencakup harga emas dalam satuan ons, gram, dan kilogram dalam USD dan IDR, serta waktu pembaruan data.

Rate Limiting

API Harga Emas memiliki batasan permintaan (rate-limiting) sebesar 100 requests per IP setiap 15 menit. Jika kamu melebihi batas ini, kamu akan menerima response 429 (Too Many Requests). Pastikan untuk menggunakan caching atau menunda permintaan untuk menghindari masalah ini.

Error Handling

Kesalahan Umum:

  • 500 Internal Server Error: Terjadi jika ada masalah dalam mengambil data dari sumber eksternal.
  • 429 Too Many Requests: Terjadi jika kamu melebihi batas permintaan sebesar 100 requests dalam 15 menit.

Jika terjadi error, API akan mengembalikan pesan error dalam format JSON:

{
    "error": "Failed to fetch data"
}
